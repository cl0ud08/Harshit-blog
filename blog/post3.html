Essential Docker Commands

  1. Check Docker Installation

    Verify if Docker is installed and running:

    docker --version
docker info

  2. Work with Docker Images

    Download, list, and remove images:

    docker pull ubuntu
docker images
docker rmi <image_id>

  3. Create and Manage Containers

    Run, stop, and remove containers:

    docker run -d -p 8080:80 nginx
docker ps
docker stop <container_id>
docker rm <container_id>

  4. Build and Push Docker Images

    Build a Docker image and push it to Docker Hub:

    docker build -t myimage:latest .
docker tag myimage:latest <your-dockerhub-username>/myimage:latest
docker login
docker push <your-dockerhub-username>/myimage:latest

  5. Manage Logs and Inspect Containers

    View container logs and inspect details:

    docker logs <container_id>
docker inspect <container_id>

  6. Work with Docker Volumes

    Create, list, and remove volumes:

    docker volume create myvolume
docker volume ls
docker volume rm myvolume

  7. Clean Up Docker System

    Remove unused containers, images, and volumes:

    docker system prune -a

  8. Docker Networks

    List and create Docker networks:

    docker network ls
docker network create mynetwork

  9. Docker Compose

    Start and stop multi-container applications:

    docker-compose up -d
docker-compose down

  10. Advanced Monitoring

    View real-time container stats:

    docker stats

  11. Docker Swarm

    Enable swarm mode and deploy services:

    docker swarm init
docker service create --name web -p 80:80 nginx
