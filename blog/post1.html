How to Deploy a Static Website on Amazon S3

Learn how to host your static website on Amazon S3 with this step-by-step guide.

  1. 1️ Create or Sign in to Your AWS Account

    Visit AWS, sign in, or create a new account.

  2. 2️ Create an S3 Bucket

    Go to AWS Console → Services → S3. Click Create Bucket, set a unique name, select a region, and create it.

  3. 3️ Enable Static Website Hosting

    In the bucket, go to Properties → Static website hosting → Enable. Set the index document as index.html.

  4. 4️ Upload Your Website Files

    In the Objects tab, click Upload, select your HTML, CSS, and assets, and confirm the upload.

  5. 5️ Make Your Files Public

    In Permissions, disable "Block all public access" and add a bucket policy for public read access.

    {
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::your-bucket-name/*"
    }
  ]
}

  6. 6️ Get Your Website URL

    Find the Bucket website endpoint in Static website hosting and visit the link to view your site.
